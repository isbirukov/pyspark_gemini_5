1. Если я выделил 10GB RAM экзекьютору, сколько реально доступно для хранения данных (Storage) по умолчанию (примерно)?
(Ответ: (10GB - 300MB) * 0.6 * 0.5 ~ 2.9 GB).

2. Что происходит, когда Execution Memory нужна память, а Storage Memory занята закэшированными данными?
(Ответ: Если данные в кэше "старые", они могут быть вытеснены, но Execution имеет приоритет, и кэш может не записаться или вылететь на диск).

3. Как увидеть, что Garbage Collector работает слишком часто?
(Ответ: UI -> Stage -> Task metrics -> GC Time красным цветом).
