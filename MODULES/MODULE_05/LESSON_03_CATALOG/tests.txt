1. Где хранятся метаданные таблиц Spark? 
(Ответ: Hive Metastore / Catalog).

2. Если я сделаю DROP TABLE для Managed таблицы, что случится с данными? 
(Ответ: Данные удалятся).

3. Если я сделаю DROP TABLE для External таблицы, что случится с данными? 
(Ответ: Данные останутся, удалится только запись в каталоге).

4. Какая SQL команда показывает, где физически лежит таблица? 
(Ответ: DESCRIBE EXTENDED table_name).
